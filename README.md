# Visionaries Platform Admin

Plataforma administrativa para Visionaries.

## Tecnologías

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Firebase

## Instalación

```bash
pnpm install
```

## Desarrollo

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Inicio

```bash
pnpm start
```
